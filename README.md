# nova-ui

## 注意事项

node 版本需要在 14 以及以上

## Development

```bash
# install dependencies
$ npm install

# develop library by docs demo
$ npm start

# build library source code
$ npm run build

# build library source code in watch mode
$ npm run build:watch

# build docs
$ npm run docs:build

# check your project for potential problems
$ npm run doctor

# 本地测试
$ npm run devPublish
```
